# Get Working Gradle Wrapper - Step by Step

## The Problem

The current `gradle-wrapper.jar` is missing the Main-Class manifest. We need a proper wrapper generated by Gradle.

---

## ✅ EASIEST SOLUTION: Download from Spring Initializr

### Step 1: Generate a Spring Boot Project

1. **Go to:** https://start.spring.io/

2. **Fill in:**
   - **Project:** Gradle - Groovy
   - **Language:** Java  
   - **Spring Boot:** 3.5.7 (or latest)
   - **Group:** com.example
   - **Artifact:** test-project
   - **Dependencies:** 
     - Spring Web
     - Spring Data JPA
     - Thymeleaf
   
3. **Click "Generate"** (green button)
   - This downloads a ZIP file

### Step 2: Extract and Copy Wrapper Files

1. **Extract the ZIP file** you just downloaded

2. **Open the extracted folder**

3. **Copy these files/folders:**
   - `gradlew` (file)
   - `gradlew.bat` (file)  
   - `gradle/` (entire folder)

4. **Paste into your project:**
   - Go to: `C:\Users\cryst\.vscode\Hack-the-Herd-F25-Wheely-Mammoth\demo`
   - **Replace/overwrite** the existing files

### Step 3: Run Your Application

```cmd
cd C:\Users\cryst\.vscode\Hack-the-Herd-F25-Wheely-Mammoth\demo
gradlew.bat bootRun
```

**This will work!** ✅

---

## ✅ ALTERNATIVE: Install Gradle

### Step 1: Download Gradle

1. Go to: https://gradle.org/releases/
2. Download: **Gradle 8.5** (Binary-only)
3. Extract to: `C:\gradle`

### Step 2: Generate Wrapper

```cmd
cd C:\Users\cryst\.vscode\Hack-the-Herd-F25-Wheely-Mammoth\demo
C:\gradle\bin\gradle.bat wrapper --gradle-version 8.5
```

### Step 3: Run

```cmd
gradlew.bat bootRun
```

---

## Why This Works

Spring Initializr generates projects with **working Gradle wrappers** that have the proper manifest. This is the fastest way to get a valid wrapper!

---

## Quick Checklist

After copying wrapper files:
- [ ] `gradlew.bat` exists
- [ ] `gradle/wrapper/gradle-wrapper.jar` exists  
- [ ] `gradle/wrapper/gradle-wrapper.properties` exists
- [ ] Run `gradlew.bat --version` (should show Gradle version)
- [ ] Run `gradlew.bat bootRun` (should start Spring Boot)
